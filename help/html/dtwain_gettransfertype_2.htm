DTwainSourceFileTransfer::GetTransferType

 Top  Previous  Next

Syntax

 

LONG DTwainSourceFileTransfer::GetTransferType( ) const;

 

Return Value

Returns the current transfer type used when doing a file transfer.  The return value will be one of the values listed in the Comments, or -1 if the Source does not support file transfers.

 

Comments

The transfer types available are below:

 

FILETYPE_SOURCE_TIFF        

TIFF Format

FILETYPE_SOURCE_PICT        

Macintosh PICT format

FILETYPE_SOURCE_BMP

Windows BMP Format

FILETYPE_SOURCE_XBM                

X Bitmap Format

FILETYPE_SOURCE_JFIF        

JFIF (JPEG) Format

FILETYPE_SOURCE_FPX        

Flash Pix Format

FILETYPE_SOURCE_TIFFMULTI        

Multi-Page TIFF

FILETYPE_SOURCE_PNG        

PNG Format

FILETYPE_SOURCE_SPIFF        

SPIFF format

FILETYPE_SOURCE_EXIF        

EXIF format
