Overview of the DTWAIN_FRAME

 Top  Previous  Next

DTWAIN provides a special data type that is to be used in place of the TWAIN TW_FRAME 'C' language structure.  This data type is the DTWAIN_FRAME.

 

When setting or getting the DTWAIN_CV_ICAPFRAMES capability values, the capability values must be stored in a DTWAIN_FRAME structure.  All values stored within the DTWAIN_FRAME are of type DTWAIN_FLOAT.
