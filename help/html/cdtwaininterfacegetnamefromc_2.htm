DTwainInterface::GetNameFromCap

 Top  Previous  Next

Syntax

static std::string DTwainInterface::GetNameFromCap( LONG Capability )

 

Parameters

Capability

The capability that is to be retrieved.

 

Return Value

If the function succeeds, the name of the capability is returned.  An empty (zero-length) std::string is returned if there is an error.

 

Comments

The DTwainInterface::GetNameFromCap function returns the string version of the capability name that is equivalent to Capability
