-verbose

 

Shows DTWAIN_TwainSave messages during acquisition.  By default, the messages are sent to the screen.  To send the messages to a file, the -verboselog option must be used in conjunction with -verbose.

 

Note that -verbose mode is not as detailed as the -diagnose option.  The -verbose mode outputs informative messages as to what DTWAIN_TwainSave is currently executing.

 

Default Setting:

No output is displayed.
