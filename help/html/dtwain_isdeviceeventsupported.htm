DTWAIN_IsDeviceEventSupported

 Top  Previous  Next

The DTWAIN_IsDeviceEventSupported determines whether a Source supports special device events.

 

DTWAIN_BOOL DTWAIN_IsDeviceEventSupported ( DTWAIN_SOURCE Source );

 

 

Parameters

Source

Specifies a TWAIN Source.

 

Return Values

If the Source supports device events, TRUE is returned, otherwise FALSE is returned.

 

Comments

The DTWAIN_IsDeviceEventSupported function returns TRUE if Source supports special device events.  Only TWAIN 1.8 compliant devices are able to support special device events.

 

TWAIN State Transitions

None.

 

Prerequisite Function Call(s)

DTWAIN_SysInitialize
