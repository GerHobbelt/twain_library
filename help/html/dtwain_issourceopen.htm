DTWAIN_IsSourceOpen

 Top  Previous  Next

The DTWAIN_IsSourceOpen function determines whether a Source is opened for acquisitions.

 

DTWAIN_BOOL DTWAIN_IsSourceOpen ( DTWAIN_SOURCE Source );

 

 

Parameters

Source

Specifies a TWAIN Source.

 

Return Values

If the function succeeds, TRUE is returned.  If the function fails FALSE is returned.

 

Comments

The DTWAIN_IsSourceOpen function returns TRUE if the Source is opened, FALSE otherwise.

An open Source is a Source that has been opened previously by selecting it, calling DTWAIN_OpenSource (TWAIN State 4 or higher), or is currently acquiring an image (State 6 and State 7).  If the Source is not opened, FALSE is returned.

 

TWAIN State Transitions

None.

 

Prerequisite Function Call(s)

DTWAIN_SysInitialize
