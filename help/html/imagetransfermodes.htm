The transfer mode defines the way the TWAIN Source will transfer the image to the application.  DTWAIN supports Native, Buffered, and File image transfer modes.

DTWAIN also has a Clipboard transfer mode that allows you to transfer an image to the Windows clipboard.  This allows your application to easily retrieve the image directly from the clipboard into many graphics programs by using the "Paste" command in the imaging application
