Acquiring Images

 Top  Previous  Next

To start the acquisition of images from the device, call one of the DTWAIN Acquisition functions.  These functions require a selected Source.

 

DTWAIN_AcquireBuffered

DTWAIN_AcquireBufferedEx

DTWAIN_AcquireToClipboard

DTWAIN_AcquireFile

DTWAIN_AcquireFileEx

DTWAIN_AcquireNative

DTWAIN_AcquireNativeEx

 

The easiest function to use successfully is DTWAIN_AcquireFile.  If you are not familiar with TWAIN, it is always best to attempt to acquire an image using this function with the format being a Windows BMP file.
