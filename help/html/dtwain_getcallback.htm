The DTWAIN_GetCallback function retrieves the function set by DTWAIN_SetCallback

 

DTWAIN_CALLBACK_PROC DTWAIN_GetCallback( );

 

Return Values

If a callback function was set using DTWAIN_SetCallback, this function returns the callback function.

 

TWAIN State Transitions

None.

 

Prerequisite Function Call(s)

DTWAIN_SysInitialize
