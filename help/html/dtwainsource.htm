[Note:  The DTwainSource section is only partially documented.  See Usage for example programs]

 

The DTwainSource class provides a high-level interface to the DTWAIN functions that deal with TWAIN Sources (reresented by a DTWAIN_SOURCE).  Wherever a DTWAIN_SOURCE is used, a DTwainSource object can be used instead.
