DTWAIN Duplex Scanner Functions

 Top 

The DTWAIN library allows you to acquire images using a Source that supports duplex scanning (scan an image on both sides of a page).

 

Check if duplex scanning is supported:

DTWAIN_IsDuplexSupported

 

Turn duplex mode on/off

DTWAIN_EnableDuplex

 

Check if duplex mode is enabled:

DTWAIN_IsDuplexEnabled

 

Get type of duplex mode that is supported:

DTWAIN_GetDuplexDevice
