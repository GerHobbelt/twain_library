-D

 

Use the duplex unit.  Default is to not use the duplex unit.  The duplex unit allows you to scan both sides of the paper.  Note that this is only available for devices that have a duplex unit.  If the device only has a duplex option, then the -D option is ignored, since there is no way to turn off the duplexer.

 

Default Setting:

The duplex unit will not be used.
