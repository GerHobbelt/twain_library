DTWAIN_GetSourceID

 Top  Previous  Next

The DTWAIN_GetSourceID returns the DTWAIN_IDENTITY of a Twain Source.

 

DTWAIN_IDENTITY DTWAIN_GetSourceID ( DTWAIN_SOURCE Source );

 

 

Parameters

Source

Specifies a selected TWAIN Source.

 

Return Values

If the function succeeds, a DTWAIN_IDENTITY of the Source is returned.  If the function fails, NULL is returned.

 

Comments

The TWAIN_IDENTITY is a pointer to a TW_IDENTITY structure that represents the Data Source's TW_IDENTITY.  See DTWAIN_GetTwainAppID for more information.   This is a Low-Level DTWAIN function that is designed to be used by advanced TWAIN programmers.

 

TWAIN State Transitions

None
