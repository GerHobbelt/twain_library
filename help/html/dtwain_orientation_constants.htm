DTWAIN Orientation Constants

 Top  Previous  Next

The DTWAIN_GetOrientation, DTWAIN_SetOrientation, and DTWAIN_EnumOrientations use and return orientation constants.  The orientation values are as follows:

 

DTWAIN Constant        

Description


DTWAIN_OR_ROT0        

Rotate 0 degrees

DTWAIN_OR_ROT90

Rotate 90 degrees

DTWAIN_OR_ROT180        

Rotate 180 degrees

DTWAIN_OR_ROT270        

Rotate 270 degrees

DTWAIN_OR_PORTRAIT        

DTWAIN_ROT0

DTWAIN_OR_LANDSCAPE

DTWAIN_ROT270
