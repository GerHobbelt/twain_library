-A

 

Use the automatic document feeder to acquire the pages.  If the TWAIN device is equipped with a document feeder, the -A option must be used to use the document feeder.  If the -A option is not specified, the flatbed portion of the device is used (if the device has a flatbed scanning option).

 

Note that some devices can only scan using a feeder, while other devices do not have feeders, therefore the -A option will have no effect on these devices.  Only if the scanner has both feeder and flatbed options will the -A option have any effect.

 

Default Setting:

Doucument feeder is not used for devices that can acquire images without a feeder.
