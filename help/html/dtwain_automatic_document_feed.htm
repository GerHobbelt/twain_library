Automatic Document Feeder Functions

 Top  Previous  Next

The following functions allow the use of a TWAIN device's automatic document feeder

 

Enable / Disable the document feeder

DTWAIN_EnableFeeder

 

Check if document feeder is enabled

DTWAIN_IsFeederEnabled

 

Check if the feeder contains any documents

DTWAIN_IsFeederLoaded

 

Check if the Source supports a document feeder

DTWAIN_IsFeederSupported

 

Check if the Source's feeder can detect if documents are loaded

DTWAIN_IsFeederSensitive
