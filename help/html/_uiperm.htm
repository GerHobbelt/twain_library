-uiperm

 

Leaves the device user interface displayed after acquiring a set of pages.  This allows the user to keep DTWAIN_TwainSave running without shutting down after a set of pages have been acquired.  The user is responsible for shutting down the user interface, thereby terminating the DTWAIN_TwainSave program.

 

The -uiperm option is not available if the -noui option is used.  If you are acquiring without a user interface, DTWAIN_TwainSave will always terminate once a set of pages have been acquired.

 

Default Setting:

The user interface is immediately closed (and DTWAIN_TwainSave is terminated) once a set of pages are acquired.
