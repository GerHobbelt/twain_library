Note that the DTWAIN library consists of many functions of almost every variety that a TWAIN application may need.  In an actual application, the application will usually utilize only a small subset of these functions.  In almost all applications, a function will be used to select the TWAIN device, and to acquire an image or set of images (as illustrated by the Simple Sample Program).

 

The following section describes all of the advanced functionality that is possible with DTWAIN that was not illustrated by the simple example.  Please read this section for information on the acquisition process, image transfer modes, device capabilities, TWAIN sessions, scaling images, setting the area to acquire, and many more topics that you may want to explore when creating your application.
