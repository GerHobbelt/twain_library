DTwainAcquirer::IsMultiPageFile

 Top  Previous  Next

Syntax

 

bool DTwainAcquirer::IsMultiPageFile( ) const;

 

 

Return Value

Returns true if the Twain Source will acquire the pages to a multi-page file type, false otherwise.

 

 

 

 

Comments

 

 

When acquiring images using DTwainAcquirer::Acquire( ) that saves the images to a file (DTWAIN File mode only), call DTwainAcquirer::SetMultiPageFile to true to acquire to a multi-page image file.  The types of files that support multi-pages are as follows:

 

FILETYPE_TIFFNONE

TIFF fornat, no compressions

FILETYPE_TIFFPACKBITS

TIFF format, packbits compressions

FILETYPE_TIFFJPEG

TIFF format, JPEG compression

FILETYPE_TIFFDEFLATE

TIFF format, Flate compression

FILETYPE_TIFFG3

TIFF format, Group 3 CCITTK compresion

FILETYPE_TIFFG4

TIFF format, Group 4 CCITTK compression

FILETYPE_PCX

PCX format

FILETYPE_PDF

Adobe Acrobat PDF format

FILETYPE_POSTSCRIPT1

Postscript Level 1 format

FILETYPE_POSTSCRIPT2

Postscript Level 2 format

FILETYPE_POSTSCRIPT3

Postscript Level 3 format

 

By default, all file acquisitions assume single page image files.

 

For example, to acquire to a multi-page PDF file:

 

DTwainSource Source;

//...

DTwainAcquirer Acq(Source);

Acq.SetMultiPageFile( true );

//...
