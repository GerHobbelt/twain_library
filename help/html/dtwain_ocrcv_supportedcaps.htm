DTWAIN_OCRCV_SUPPORTEDCAPS
 

 Description

Returns a list of all of the capabilities that are supported by the OCR engine.

 

 Values

No Default.

 

Allowed Values:

Not Applicable.

 

Available Operations:

DTWAIN_CAPGETCURRENT

 

Mandatory Capability:
