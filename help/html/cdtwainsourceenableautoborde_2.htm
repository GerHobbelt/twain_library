DTwainSource::EnableAutoBorderDetect

 Top  Previous  Next

Syntax

bool EnableAutoBorderDetect( bool bEnable = true ) const;

 

Parameters

bEnable

true enables duplex mode, false disables duplex mode.

 

Return Value

true if successful, false if unsuccessful.

 

Comments

DTwainSource::EnableAutoBorderDetect enables or disables the Twain drivers autoborder detect mode depending on the bEnable parameter.  If bEnable is true, then auto-border detect is turned on, else it is turned off.

 

To test if the Twain Source supports autoborder detection, call DTwainSource::IsAutoBorderDetectSupported.
