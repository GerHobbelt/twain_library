Advanced DTWAIN Implementation

 Top 

The following steps describe how a typical application can easily incorporate TWAIN with most of the advanced settings available. These steps allow you to

 

•Choose whether the application or DTWAIN controls the TWAIN message loop.
•Start a TWAIN Session
•Set Device capabilities
•Set the Image page size to acquire
•Capture DTWAIN messages and notifications
•Acquire multiple pages

 

There are many more advanced features in DTWAIN.  See the topic DTWAIN Power User Information
