DTwainSource::EnumPixelTypes

 Top  Previous  Next

Syntax

bool DTwainSource::EnumPixelTypes( DTwainLongArray& Array ) const;

 

Parameters

 

Array

The DTwainLongArray that will store the supported pixel types on return.

 

Return Value

 

true if successful, false otherwise.

 

Comments

This function fills the DTwainLongArray Array with the available pixel types.  The pixel type determines the color used by the acquired image.
