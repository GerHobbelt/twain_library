Data Types used by DTWAIN

 Top 

The following data types are used by DTWAIN.

 

Windows Types

 

HANDLE

LONG

LPVOID

 

DTWAIN Types

 

LPDTWAIN_ARRAY

DTWAIN_ARRAY

DTWAIN_BOOL

DTWAIN_FLOAT

DTWAIN_RANGE

DTWAIN_STRING

DTWAIN_SOURCE

 

All integer data types (signed, unsigned, 16 and 32 -bit) are of type LONG (32-bits).  This allows a uniform integer type that is supported by most platforms. Fatal errors can occur if the application language assumes 16-bit integers while DTWAIN is using 32-bit integers.  For example, DTWAIN_ARRAYs always uses LONGs internally and expects the application to get and retrieve values from the DTWAIN_ARRAY using LONGs.
