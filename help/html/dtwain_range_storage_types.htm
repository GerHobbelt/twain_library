DTWAIN_RANGE Storage Types

 Top  Previous  Next

The DTWAIN_RANGE can store only two types of data; LONGS and DTWAIN_FLOATs

 

Data type

DTWAIN_RANGE constant

LONG        

DTWAIN_RANGELONG

DTWAIN_FLOAT

DTWAIN_RANGEFLOAT

 

Whenever your application requests a DTWAIN_RANGE, the application must specify the type of data the range will store.  The DTWAIN_RANGE constants defined above are used to specify the type.
