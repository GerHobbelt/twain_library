DTwainAcquirer::EnableDuplex

 Top  Previous  Next

Syntax

 

DTwainAcquirer& DTwainAcquirer::EnableDuplex( bool bEnable = true );

 

Parameters

bEnable

 

Enables the device's duplex unit if bEnable is true.  Turns off the duplex unit if bEnable is false.

 

Return Value

Returns reference to current DTwainAcquirer object (*this).
