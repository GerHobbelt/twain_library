Color Lamp Selection

 Top  Previous  Next

Some devices have the ability to select the color lamp to use when acquiring images.   The color lamp is also known as the light source.

 

The following functions test to see if the TWAIN device supports different light sources, and allow setting of the various light sources.

 

Testing for support

DTWAIN_IsLightSourceSupported

DTWAIN_EnumLightSources

 

Enabling a light source

DTWAIN_SetLightSources

 

Retrieving Current Light Source

DTWAIN_GetLightSources
