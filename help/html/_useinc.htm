-useinc

 

Turns on the smart auto-increment feature of DTWAIN_TwainSave when saving multiple pages to separate files, or when saving separate jobs to files using the -jc command.

 

See the -f option for more details on how the -useinc option names files when saving multiple files or jobs.

 

Default Setting:

The smart auto-increment feature is turned off.
