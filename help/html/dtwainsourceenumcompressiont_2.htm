DTwainSource::EnumCompressionTypes

 Top  Previous  Next

Syntax

bool DTwainSource::EnumCompressionTypes( DTwainLongArray& Array ) const;

 

Parameters

Array

On return, stores the supported compression types used by the buffered transfer mode.

 

Return Value

true if successful, false otherwise.

 

Comments

This function returns a DTwainLongArray that denotes all of the supported compression types when transferring an image using the buffered transfer.  See the main help manual for the various compression type constants.
