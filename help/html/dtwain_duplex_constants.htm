DTWAIN Duplex Constants

The DTWAIN_GetDuplexType returns one of the following values that denotes the type of duplex that is supported.

 

DTWAIN Constant

Description


DTWAIN_DX_NONE        

No duplexer found.

DTWAIN_DX_1PASSDUPLEX

One pass duplexer

DTWAIN_DX_2PASSDUPLEX

Two pass duplexer
