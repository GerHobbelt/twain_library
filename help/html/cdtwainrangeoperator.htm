DTwainRange::operator =

 Top  Previous  Next

Syntax

DTwainRange& operator = ( const DTwainRange& Range );

DTwainRange& operator = ( DTWAIN_RANGE Range );

 

Parameters

Range

An existing CDTwainString object, or a DTWAIN_RANGE.

 

Return Value

Returns the current object.

 

Comments

Copies an existing DTwainRange object Range.  A DTWAIN_RANGE can also be used for Range.
