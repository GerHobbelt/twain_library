DTwainSource::EnableAutoDeskew

 Top  Previous  Next

Syntax

bool EnableAutoDeskew( bool bEnable = true ) const;

 

Parameters

bEnable

true enables auto-deskew mode, false disables auto-deskew mode.

 

Return Value

true if successful, false if unsuccessful.

 

Comments

DTwainSource::EnableAutoDeskew enables or disables the Twain drivers auto-deskewt mode depending on the bEnable parameter.  If bEnable is true, then auto- deskew is turned on, else it is turned off.

 

To test if the Twain Source supports autoborder detection, call DTwainSource::IsAutoDeskewSupported.
