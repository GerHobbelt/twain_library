DTwainCompressedTransfer Overview

 Top  Previous  Next

The DTwainCompressedTransfer class contains functions that determine if a DTwainSource supports acquiring images using various compression options.  The DTwainCompressedTransfer is used as a base class to the DTwainBufferedTransfer and DTwainSourceModeTransfer classes, since each of these classes support compressed image retrieval.  Usage of this class allows your application to set up compressed image retrieval with full knowledge of what types of compression exists with the Twain Source.

 

Note that the application cannot create DTwainCompressedTransfer objects directly, since the constructor is protected.  Instead the application must create a DTwainBufferedTransfer or DTwainSourceModeTransfer object to utilize the DTwainCompressedTransfer functions.
