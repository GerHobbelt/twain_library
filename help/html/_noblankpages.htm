-noblankpages        

 

The -noblankpages flag allows DTWAIN_TwainSave to not save blank pages when acquiring pages from the TWAIN device.  If the page being acquired is considered blank, then page is not saved to the image file.

 

Also see the -blankthreshold option to set the threshold value to determine if a page is blank.  If -blankthreshold is not used, the threshold value is 98.

 

Please note that detection of blank pages may slow down the acquisition of data from the TWAIN device.

 

Default Setting:

Blank pages are saved to the image file.
