DTwainInterface::GetAppInfo

 Top  Previous  Next

Syntax

bool DTwainInterface::GetAppInfo( DTwainAppInfo& AppInfo )

 

Parameters

AppInfo

On successful return, the current TWAIN application information will be copied to the DTwainAppInfo object.

 

Return Value

true if successful, false otherwise.

 

Comments

This function retrieves application information from the TWAIN Source Manager and fills the DTwainAppInfo object with the information.
