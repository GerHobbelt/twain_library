-C value

 

Sets the contrast level of the image that is being acquired. The device must support setting constrast values.  Some devices do not support setting the constrast level.  The -verbose option will generate a message that will inform whether setting the constrast worked successfully or an error was encountered.

 

The constrast value can range from -1000 to 1000.  The lower the number, the less constrast the image will be when acquired.  Also see the -B command to set the brightness of the image.

 

To set the contrast to a middle value:

-C 0

 

To set the contrast to the higheset level:

-C 1000

 

Default Setting:

The contrast value that the device determines as the default.  Usually this is a value of 0.
