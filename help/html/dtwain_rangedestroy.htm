DTWAIN_RangeDestroy

 Top  Previous  Next

The DTWAIN_RangeDestroy function destroys a DTWAIN_RANGE.

 

DTWAIN_BOOL DTWAIN_RangeDestroy ( DTWAIN_RANGE Range );

 

 

Parameters

Range

DTWAIN_RANGE that will be destroyed.

 

Return Values

If the function succeeds, TRUE is returned.  If the function fails FALSE is returned.

 

Comments

The DTWAIN_RangeDestroy function destroys the DTWAIN_RANGE Range and frees any resources allocated to Range.

 

TWAIN State Transitions

None.

 

Prerequisite Function Call(s)

DTWAIN_SysInitialize
