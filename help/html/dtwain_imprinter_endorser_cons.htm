DTWAIN Imprinter/Endorser Values

 Top  Previous  Next

When using an imprinter or endorser, (DTWAIN_SetPrinter, DTWAIN_GetPrinter, etc.), the following constants describe the types of imprinter and endorsers that are available.  The Definition column describes where the imprinter/endorser is located (Top or Bottom), and whether it prints before or after an image is acquired (Before/After).

 

Imprinter/Endorser Type

Definition


DTWAIN_IMPRINTERTOPBEFORE        

Top Imprinter, Before scan

DTWAIN_IMPRINTERTOPAFTER                

Top Imprinter, After scan

DTWAIN_IMPRINTERBOTTOMBEFORE        

Bottom Imprinter, Before scan

DTWAIN_IMPRINTERBOTTOMAFTER

Bottom Imprinter, After scan

DTWAIN_ENDORSERTOPBEFORE        

Top Endorser, Before scan

DTWAIN_ENDORSERTOPAFTER                

Top Endorser, After scan

DTWAIN_ENDORSERBOTTOMBEFORE        

Bottom Endorser, Before scan

DTWAIN_ENDORSERBOTTOMAFTER

Bottom Endorser, After scan

 

 

The Imprinter/Endorser string functions (DTWAIN_GetPrinterStringMode, DTWAIN_SetPrinterStringMode. etc.) use the following constants to define the various string modes:

Imprinter/Endorser String Type

Definition


DTWAIN_PM_SINGLESTRING        

Single String

DTWAIN_PM_MULTISTRING        

Multiple Strings

DTWAIN_PM_COMPOUNDSTRING

Compound String
