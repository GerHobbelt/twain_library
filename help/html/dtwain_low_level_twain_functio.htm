Low-Level TWAIN Functions

 Top  Previous  Next

The following functions allow an application to make calls directly to the TWAIN Data Source Manager.  Calling the TWAIN Data Source Manager directly should be done by experienced programmers (most likely C or C++ programmers) who have knowledge of the TWAIN API and Data Source Manager.  

 

Using DTWAIN, there is usually no need to call these functions, however they are provided in case there are certain aspects of the TWAIN API that may not have been handled by DTWAIN.

 

Get the TW_IDENTITY values

DTWAIN_GetTwainAppID

DTWAIN_GetSourceID

 

Call TWAIN Data Source Manager (DSM) directly using a Twain Triplet

DTWAIN_CallDSMProc
