-U measure        

 

Sets the unit of measure of the device.  Used with the -R command.  The values of measure are as follows:

 

Measurement

Description

 

inch

Inches

cm

Centimeters

twip

1440 twips per inch

pica

6 picas per inch

pixel

Current pixel width of device

point

72 points per inch

 

Note: The TWAIN device must support the unit of measurement.

 

If the -area command is used, the unit of measure is the unit specified in the -area command

 

Default Setting:

inch
