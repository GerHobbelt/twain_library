Multi-Page Acquisitions

 Top  Previous  Next

DTWAIN supports multi-page acquisitions with virtually no extra work for your application.

 

The TWAIN specification states that the application is required to set certain device capabilities to enable multi-page scanning.  DTWAIN does all of this work for you automatically; there is no need for your application to set any special capabilities to handle scanning more than one page.  

 

If your image device is equipped with an automatic document feeder (ADF), DTWAIN allows you to scan all of the pages, or set the maximum number of pages to acquire (this is good for those applications that want to acquire only one page, even though the device may support multiple pages).  There are also functions to determine whether to use the ADF or flatbed portion of the device, and to check if there are documents loaded in the feeder.

 

Also, if you have a device that only acquires single pages (for example, a flatbed scanner), you can still acquire to multipage image files, even though you can only scan a page at a single time.  DTWAIN can collate the scanned pages into a single multipage image file.
