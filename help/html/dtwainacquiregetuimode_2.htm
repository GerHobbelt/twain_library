DTwainAcquirer::GetUIMode

 Top  Previous  Next

Syntax

 

bool DTwainAcquirer::GetUIMode( ) const;

 

 

Return Value

Returns true if the Twain Source will use the Source's user interface when acquiring images, false otherwise.

 

 

 

 

Comments

 

 

To set the Source's user interface mode, call DTwainAcquirer::SetUIMode.
