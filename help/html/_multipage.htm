-multipage

 

Save all pages scanned to a multi-page file.  This switch can only be used for the following file types (see the -F parameter):

 

pcx

PCX Format

pdf

Adobe PDF format

ps1

PostScript Level 1

ps2

PostScript Level 2

tif1

TIFF (no compression)

tif2

TIFF (PackBits compression)

tif3

TIFF (Group 3 FAX compression)

tif4

TIFF (Group 4 FAX compression)

tif5

TIFF (JPEG compression)

tif6

TIFF (Flate compression)

tif7

TIFF (LZW compression)

 

Default Setting:

No multipage (all acquisitions are single pages).
