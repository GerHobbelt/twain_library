-G value        

 

Sets the gamma correction value for the images to be acquired. The device must support setting gamma values.  Some devices do not support setting the gamma level.  The -verbose option will generate a message that will inform whether setting the gamma correction worked successfully or an error was encountered.

 

The gamma value can be any numeric values.

 

Default Setting:

The gamma value that the device determines as the default.  Usually this is a value of 2.2.
