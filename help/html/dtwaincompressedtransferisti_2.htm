DTwainCompressedTransfer::IsTiffCompressionSupported

 Top  Previous  Next

Syntax

 

bool DTwainCompressedTransfer::IsTiffCompressionSupported( ) const;

 

 

 

Comments

 

 

Returns true if the attached Twain Source supports any TIFF compression (Group3, Group4, LZW, etc.).  The return value is false if TIFF compression is not supported, or if there is an error.
