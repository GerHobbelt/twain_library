Installation/Components of DTWAIN

 Top  Previous  Next

The installation procedure creates a directory tree for the DTWAIN files, and adds the demonstration programs and online help to your desktop. You specify the root directory during installation.

 

The following is a list of the subdirectories in the root directory:

Bin

Contains GetVersion.exe program that can be used to test what version of DTWAIN32.DLL will be accessible when running a program.

Borland C++\5.0\Lib\MultiThread

Import Library for Borland C++ 5.02c

 

Borland C++Builder\Lib\MultiThread

Import Library for Borland (Inprise) C++ Builder 5.0 and above.

 

CPP_Classes

 

C++ wrapper for DTWAIN

CSharp

C# Interface file (DTWAIN32.CS, DTWAIN64.CS).for DTWAIN

 

Delphi

Delphi .PAS file for DTWAIN

 

DLL

32-bit versions of DTWAIN32.DLL, DTWAIN32U.DLL, TWAINDSM.DLL, plus language DLL's.

 

DLL_64

64-bit versions of the DTWAIN libraries (DTWAIN64.DLL, DTWAIN64U.DLL, TWAINDSM.DLL)

 

DYDTWAIN

Contains C and C++ source files that implements the usage of Windows LoadLibrary( ) function to load DTWAIN32.DLL at runtime.

 

EXAMPLES\C

WIN 32/64 API 'C' language examples.

 

EXAMPLES\CPP

WIN 32/64 API C++ language examples.

 

EXAMPLES\CSharp

 

C# Examples.

EXAMPLES\DELPHI

Delphi Examples

 

EXAMPLES\DIBVIEW

Add on C source code for C and C++ examples

 

EXAMPLES\MFC

Win 32/64 MFC examples

 

EXAMPLES\VB.NET

Visual Basic .NET examples.

 

EXAMPLES\VisualBasic

Visual Basic examples.

 

FoxPro

Visual FoxPro DTWAIN interface files.

 

HELP

Online help files.

 

INCLUDE

Header files for C and C++ development

 

INI

DTWAIN32.INI customization file

 

License

DTWAIN License Agreement.

 

MacroScheduler

Macro Scheduler interface files.

 

Perl

Perl language interface files.

 

Python

Python language interface files.

 

README        

Last minute DTWAIN information not included in this help file.

 

TESTBED

Scanner / Digital Camera Twain test programs written with DTWAIN

 

TESTBED64

64-bit version of Testbed applications.

 

VisualBasic\BAS

Visual Basic 4.x and above .BAS files

 

VisualBasic\Net

Visual Basic .NET interface files.

 

Visual C++\5.0\Lib

Import Libraries for Visual C++ 5.0 MultiThread

 

Visual C++\6.0\Lib

Import Libraries for Visual C++ 6.0\Net Multithread

 

WinBatch

WinBatch interface files.

 

XBase++

XBase++ interface files.
