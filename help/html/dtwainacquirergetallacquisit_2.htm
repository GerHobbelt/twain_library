DTwainAcquirer::GetAllAcquisitions

 Top  Previous  Next

Syntax

 

 

bool DTwainAcquirer::GetAllAcquisitions( DTwainAcquisitionArray& Array );

 

 

Return Value

Returns true if successful, false otherwise.

 

 

 

 

Comments

 

 

The DTwainAcquirer::GetAllAcquisitions returns an array Array of all the acquired image data from the previous Twain session whenever a native or buffered transfer is done.  The image data is stored in a DTwainAcquisitionArray.  Each element of the DTwainAcquisitionArray is itself a DTwainImageDataArray, which is an array of DTwainImageData.

 

This effectively making a DTwainAcquisitionArray a two-dimensional array (i.e. Array[Acquisition][page].GetImageData( ) is the actual image data for a particular acquisition and page number).

 

See Example 4 for usage of DTwainAcquirer::GetAllAcquisitions.
