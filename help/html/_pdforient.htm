-pdforient Orientation

 

The -pdforient option sets the orientation for PDF pages.  The value for Orientation must be either portrait or landscape.  By default, the orientation used in a PDF document is portrait.

 

Example:

 

DTWAIN_TwainSave("-f pdf -pdforient landscape");

 

Sets the orientation to landscape.

 

Default Setting:

Portrait
